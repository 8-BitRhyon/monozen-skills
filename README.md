# monozen-skills — Agent Repertoire

Skills for the Monozen portfolio system (dual-persona SPA, Sun/Shaper & Moon/Systemizer).

## Install

```bash
npx skills add 8-BitRhyon/monozen-skills --skill <name>
```

## Skills

| Skill | Trigger |
|-------|---------|
| `monozen-architecture` | File structure, panel system, theme lifecycle, CSS architecture |
| `monozen-themes` | Sun/Shaper vs Moon/Systemizer identity, tokens, motion contract |
| `monozen-webgl` | __MzGL pipeline, shaders, concurrent loop prevention, Paper Shaders |
| `monozen-nav` | Capsule layout, corner brackets (GSAP Flip), brand crossfade, responsive |

## Stack

Zero dependencies. SKILL.md files installable via [Agent Skills](https://github.com/vercel-labs/skills).
